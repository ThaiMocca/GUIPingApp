# Ping Application
## Purpose
This application was developed as part of an assignment in the course IIA2017 at the University of South-Eastern Norway. Its primary purpose is to demonstrate different methods of pining a selected network node.
## Functionality
The tool provides four ping modes: Single, Fixed Attempts, Continuous and Broadcast. Each mode incldues its own set of parameters that must the configured by the user. The results of each ping operation are displayed in the GUI's output section.
